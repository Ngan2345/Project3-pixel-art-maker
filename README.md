# Pixel Art Maker

This is **Project 3** from **Udacity's Intro to Programming Nanodegree Program**.

## 1. Description
In this project, I was provided with the [starter code](https://github.com/udacity/project-pixel-art-maker-starter) to build a single-page app allowing users to draw pixel art on a customisable canvas.


## 2. Language
- JavaScript   

## 3. How to use this project

- Download `index.html`, `styles.css`and `designs.js` files to your computer and store them in the same folder
- Open `index.html` in your browser
- Choose the size for your canvas and pick a colour to start designing your art
